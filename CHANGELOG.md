# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-29

### Added
- Initial release of MXW01 Thermal Printer addon
- Bluetooth connectivity to MXW01 printer
- Text printing with customizable font sizes
- Image printing from local files and URLs
- Web UI for manual printer control
- Auto-connect functionality on startup
- Configurable print intensity (0-255)
- Multiple dithering methods (Floyd-Steinberg, Bayer, Atkinson, None)
- REST API for Home Assistant integration
- Service definitions for automation support
- Test print functionality
- Real-time printer status display
- Support for multiple architectures (amd64, aarch64, armhf, armv7, i386)
