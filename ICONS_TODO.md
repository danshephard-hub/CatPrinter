# Addon Icons TODO

The addon requires two icon files for proper display in Home Assistant:

## Required Files

1. **icon.png** - 256x256 pixels
   - Used as the addon icon in the add-on store
   - Should represent a thermal printer

2. **logo.png** - 128x128 pixels
   - Used as the addon logo
   - Should be a smaller version of the icon

## Creating the Icons

You can create these icons using:
- Any image editor (GIMP, Photoshop, etc.)
- Online icon generators
- Use an existing printer icon from open-source icon sets

## Recommended Design

- Simple, clean printer icon
- Consider using a thermal receipt coming out of a printer
- Use colors that match the Home Assistant theme
- Ensure good contrast for visibility

## Placement

Place the created files in the root directory:
```
/home/dan/Documents/CatPrinter/icon.png
/home/dan/Documents/CatPrinter/logo.png
```
